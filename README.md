# 🚀 CRYPTOPORT – AI Powered Crypto Portfolio Manager

## 📌 Overview

CRYPTOPORT is an AI-powered cryptocurrency portfolio management platform developed using Python and Streamlit. The application helps users track cryptocurrency investments, monitor real-time market trends, analyze portfolio performance, evaluate investment risks, and generate AI-based trading recommendations.

The platform integrates real-time cryptocurrency market data, forecasting analytics, portfolio risk assessment, interactive dashboards, and AI-assisted insights to support informed investment decisions.

---

## ✨ Features

* 📈 Real-Time Cryptocurrency Market Monitoring
* 💼 Portfolio Management and Tracking
* 🤖 AI-Based Buy, Hold, and Sell Recommendations

* ⚠️ Portfolio Risk Analysis

## 📊 Modules

### 1. Real-Time Market Analysis

Fetches and displays live cryptocurrency prices and market trends.

### 2. Portfolio Management

Tracks holdings, investment value, and profit/loss performance.

### 3. AI Trading Signals

Generates Buy, Hold, and Sell recommendations.

### 4. Forecasting System

Predicts future cryptocurrency market trends.

### 5. Risk Analysis

Calculates portfolio volatility and investment risk scores.

### 6. Interactive Dashboard

Provides dynamic charts and financial visualizations.

### 7. AI Assistant

Answers cryptocurrency-related questions and provides portfolio insights.
